# Klik PRO 1.4.0

Klik PRO 1.4.0 refreshes the main window navigation and the Mappings screen.

- The tab bar is now a pill-shaped control: the active tab is a filled-blue pill,
  tabs are evenly spaced, and the row is centered in the window header and raised
  beside the logo. Tab order is Mappings, App Profiles, Settings, Advanced.
- The **Check for Updates** button is tightened to sit cleanly within the new header
  layout.
- The Mappings **Native Apps** card shows a loading spinner during the first-launch
  app scan instead of flashing "No native apps installed" over empty space.
- Mappings cards auto-hide their scrollers and size their content accurately: a group
  that fits shows no near-full stub handle, and an overflowing list gets a
  proportional handle.
- **Thumb Wheel Tab Switching** has moved from Mappings to the Settings tab, grouping
  it with the app's other preferences.

Tab indices and hit-testing are unchanged, so existing Thumb Wheel and keyboard
navigation continue to work as before.
