# Klik PRO 1.4.2

Klik PRO 1.4.2 brings the App Profile generator card up to full parity with the
per-profile controls for the native Klik PRO Dock launcher.

- The generator card's gear menu gains **Rename Dock Icon…** and **Change Icon…**,
  acting on the native app's Klik PRO Dock launcher. Rename updates the Dock tile
  label; Change Icon supports tint, badge, a custom PNG/ICO image, and reset — the
  same options already available on each managed profile.
- Both the custom name and the custom icon are **durable**: they survive an App
  Profile generation and a gear **Replace**, so a launcher you have personalized
  keeps its name and icon across rebuilds.

Because the Dock and LaunchServices cache icons and labels, a change may take a
moment — or a relaunch — to appear on the tile.

No other App Profile behavior changes in this release.
