# Klik PRO 1.4.1

Klik PRO 1.4.1 is a small front-of-app polish release.

- The tab bar now has a subtle border and sits on the header's centerline, aligned
  with the Klik PRO wordmark and the updates button. Each tab's label — and the
  Advanced lock glyph — is vertically centered from measured text metrics, so the
  row reads as one evenly aligned control.
- The header's **Check for Updates** button is now a compact **↻ Updates…** control,
  narrowed to fit while keeping its top-right position. When a newer version is
  found it still reads **Update available**.
- Wording for the built-in ChatGPT and Claude apps is now consistently "native" (for
  example the menu-bar entry reads "ChatGPT / Codex (native)"), matching the rest of
  the App Profiles screen.

No functional or App Profile behavior changes in this release.
